### Chapter 2 Algorithms
1. 算法的特性：输入、输出、有穷性、确定性和可行性
2. 算法的特征：正确性、可读性、健壮性、高效率和低存储量
3. 根据数学中函数的特点来类比算法：
   1. 判断一个算法的效率时，函数中的常数和其他次要项常常可以忽略，而更应该关注主项（最高阶项）的阶数。
   2. 某个算法，随着n的增大，它会越来越优于另一算法，或者越来越差于另一算法
